# Gestor de archivos 

Sistema web tipo CRUD para gestión de archivos

## Tecnologías utilizadas 

-PHP
-MySQL
-JavaScript
-HTML/CSS
-Bootstrap

## Funcionalidades 
-Gestión de archivos 
-Carga y descarga de documentos
-Administración de usuarios
-Organización de información
-Envio de notificaciones
